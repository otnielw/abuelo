Breadth-First Search

Explores a graph level by level, visiting all neighbors of the current node before moving deeper. Uses a queue (FIFO) to determine traversal order. Guarantees the shortest path in unweighted graphs.

Time O(V + E) Space O(V)

2
Depth-First Search

Explores as far as possible along each branch before backtracking. Uses a stack (LIFO) or recursion to track which node to visit next. Useful for topological sorting, cycle detection, and maze solving.

Time O(V + E) Space O(V)
Current On Stack Visited

Step 2

Pop top node, mark visited

Step 3

Push all unvisited neighbors

Step 4

Backtrack when no unvisited neighbors remain

3
Dijkstra's Shortest Path

Finds the shortest path from a source to all other nodes in a weighted graph with non-negative edge weights. Uses a priority queue to always explore the closest unvisited node next.

Time O((V+E) log V) Space O(V)
Current Relaxing Finalized
Speed
Press Find Paths to begin from node A
Step 1

Set source distance to 0, all others to infinity

Step 2

Pick unvisited node with smallest distance

Step 3

Update distances of its neighbors (relax edges)

Step 4

Mark node as finalized, repeat until all visited

4
Prim's Minimum Spanning Tree

Builds a minimum spanning tree by starting from any node and greedily adding the cheapest edge connecting a new node to the growing tree. Produces a tree connecting all nodes with minimum total weight.

Time O((V+E) log V) Space O(V)
Current Candidate In Tree
Speed
Press Build MST to begin from node A
Step 1

Start from any node, add to tree

Step 2

Find cheapest edge from tree to non-tree node

Step 3

Add that edge and node to tree

Step 4

Repeat until all nodes are in tree

5
A* Search

An informed search algorithm that uses a heuristic to guide pathfinding. Combines Dijkstra's guarantee of shortest path with greedy best-first search's speed. Widely used in games and robotics.

Time O(E) Space O(V)

Speed
Click cells to place walls, then press Find Path
Step 1

Add start to open set with f = g + h

Step 2

Pick node with lowest f-score from open set

Step 3

Explore neighbors, update g-scores, compute h (Manhattan)

Step 4

Reconstruct path when end node is reached
